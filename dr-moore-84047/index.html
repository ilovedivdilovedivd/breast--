Why does only acetone help chemical peels for my skin?
Using acetone for chemical peel
Sep 16, 2019
The use of acetone should always be incorporated into the chemical peel treatment. It allows your skin to fully degrease so that the peel can drive deeper into the skin and become more effective. If you are using isopropyl it will still clean the skin but not fully degrease it so the peel might not drive as deep and not be as effective. Best wishes!

Best wishes!
